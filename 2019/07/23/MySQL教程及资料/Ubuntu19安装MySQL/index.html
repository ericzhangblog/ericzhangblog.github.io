3、配置MySQL

3.1 配置初始化信息

1.sudo mysql_secure_installation

配置说明：





3.2 配置访问权限

在Ubuntu下MySQL缺省是只允许本地访问的，使用Navicat等连接工具是连不上的；
如果你要其他机器也能够访问的话，需要进行配置；

3.2.1 配置远程访问
登陆mysql
    sudo mysql -uroot -p
执行下句话，表示授权
    (1)mysql> GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY '自己设置的密码' WITH GRANT OPTION; 
-->   %:表示从任何主机连接到mysql服务器
    (2)mysql> FLUSH PRIVILEGES;

4 MySQL服务命令

# 查看服务状态：
sudo service mysql status
# 停止
sudo service mysql stop
# 启动
sudo service mysql star
